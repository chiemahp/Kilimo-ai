# Kilimo AI Workflow 🔄

## User Journey
1. **Input**: User enters agricultural query in frontend.
2. **Processing**: FastAPI routes request to Jaseci agents.
3. **Agent Logic**: JacLang agents evaluate crop/soil/climate data.
4. **Output**: Insights returned to frontend for display.

---

## Developer Workflow
- **Frontend Dev**: Build UI components in React/Vite.
- **Backend Dev**: Add routes/services in FastAPI.
- **Agent Dev**: Extend JacLang agents with new agricultural logic.
- **Deployment**: Use Docker Compose for local or cloud deployment.
