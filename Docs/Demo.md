# Kilimo AI Demo 🎥

## 1. Introduction
Kilimo AI empowers farmers with AI-driven agricultural insights. It combines FastAPI, React/Vite, and Jaseci agents to deliver modular, scalable solutions.

---

## 2. Features
- **Frontend UI**: Clean, modern interface built with React/Vite.
- **Backend API**: FastAPI routes for secure, modular communication.
- **AI Agents**: JacLang agents providing crop, soil, and climate advice.
- **Deployment**: Dockerized setup for easy scaling.

---

## 3. Workflow
1. User submits a query via the frontend.
2. Request is routed to FastAPI backend.
3. Backend invokes Jaseci agents.
4. Agents process agricultural logic and return insights.
5. Response is displayed in the frontend UI.

---

## 4. Jaseci Components
- **Agents**: Encapsulate agricultural decision logic.
- **Sentinel**: Manages agent orchestration.
- **Walker**: Executes workflows.
- **Graph**: Represents knowledge structures (crop → soil → climate).
